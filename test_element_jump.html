Element Jump Functionality Test

Description

This test demonstrates the element positioning jump function that can navigate to specific elements by ID in draw.io diagrams.

The functionality allows:

  • Jumping to an element by its ID on the drawing
  • Using URL parameters to jump after loading the drawing

URL Parameter Format

To use this functionality, add the elementId parameter to your URL:

http://your-domain.com/drawio/?elementId=YOUR_ELEMENT_ID

Example with other parameters:

http://your-domain.com/drawio/?url=diagram.drawio&elementId=shape123

JavaScript API

You can also call the function programmatically:

// Jump to an element by ID
jumpToElement(graph, 'your-element-id');

// The function returns true if element is found, false otherwise
var success = jumpToElement(graph, 'element123');
if (!success) {
    console.log('Element not found');
}
