## 测试用例

### 用例1: 单一连接类型
**输入数据:**
```
互联起点: T1
互联终点: T2  
互联类型: 刀开关
```

**预期结果:**
- 创建中间节点（刀开关图形）
- 两条边: T1→刀开关, 刀开关→T2

**实际结果:** ✅ 符合预期

### 用例2: 空连接类型
**输入数据:**
```
互联起点: T3
互联终点: T4
互联类型: (空)
```

**预期结果:**
- 不创建中间节点
- 直接连线: T3→T4

**实际结果:** ✅ 符合预期

### 用例3: 混合连接
**输入数据:**
```
1. T1 → T2 (刀开关)
2. T2 → T3 (空)
3. T3 → T4 (LED)
```

**预期结果:**
- 3条边，2个中间节点
- 正确的连接序列

**实际结果:** ✅ 符合预期
