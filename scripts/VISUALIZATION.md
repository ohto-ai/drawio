# 互联类型功能 - 可视化说明

## 功能对比

### 传统方式（无互联类型）

```
Excel数据:
  互联起点: Terminal_A
  互联终点: Terminal_B
  互联类型: (空)

生成结果:
  Terminal_A ──────────────── Terminal_B
             (直接连线)
```

### 新方式（使用互联类型）

```
Excel数据:
  互联起点: Terminal_A
  互联终点: Terminal_B
  互联类型: 刀开关

生成结果:
                    ╔═══════╗
  Terminal_A ───────║ 刀开关 ║─────── Terminal_B
                    ║  ⚡   ║
                    ╚═══════╝
           (自动插入中间节点)
```

## XML结构对比

### 无互联类型的边

```xml
<!-- 直接连线 -->
<mxCell id="E1" edge="1" parent="1" 
        source="Terminal_A" 
        target="Terminal_B">
    <mxGeometry relative="1" as="geometry"/>
</mxCell>
```

### 有互联类型的边

```xml
<!-- 中间节点（刀开关） -->
<mxCell id="N1" value="" vertex="1" parent="1"
        style="html=1;shape=mxgraph.electrical.electro-mechanical.singleSwitch;...">
    <mxGeometry x="0" y="0" width="75" height="20" as="geometry" relative="1"/>
</mxCell>

<!-- 第一条边：Terminal_A → 刀开关 -->
<mxCell id="E1" edge="1" parent="1" 
        source="Terminal_A" 
        target="N1">
    <mxGeometry relative="1" as="geometry"/>
</mxCell>

<!-- 第二条边：刀开关 → Terminal_B -->
<mxCell id="E2" edge="1" parent="1" 
        source="N1" 
        target="Terminal_B">
    <mxGeometry relative="1" as="geometry"/>
</mxCell>
```

## 图形元件库

### 当前支持的元件

```
┌──────────┬──────────────────────────┬────────┐
│ 类型名称 │ 图形描述                 │ 尺寸   │
├──────────┼──────────────────────────┼────────┤
│ 刀开关   │ ⚡ 单刀开关（断开状态） │ 75×20  │
│ 闭合开关 │ ⚡ 单刀开关（闭合状态） │ 75×20  │
│ 双刀开关 │ ⚡⚡ 双刀双掷开关        │ 75×40  │
│ 压板     │ ⊏━━━⊐ 压板连接器        │ 60×20  │
│ LED      │ 💡 LED指示灯            │ 40×40  │
└──────────┴──────────────────────────┴────────┘
```

### 添加自定义元件

在 `terminal_data_model.py` 中修改 `COMPONENT_GRAPHICS`:

```python
COMPONENT_GRAPHICS = {
    # ... 现有类型 ...
    
    # 添加新类型
    "继电器": {
        "style": "html=1;shape=mxgraph.electrical.electro-mechanical.relay;",
        "width": 60,
        "height": 40,
        "value": ""
    },
}
```

然后在Excel中即可使用：

```
互联类型: 继电器
```

## 使用场景

### ✅ 适用场景

1. **电气接线图**: 开关、继电器、指示灯等元件
2. **控制回路**: 需要显示控制元件的连接
3. **保护回路**: 保护装置、跳闸回路等
4. **信号回路**: 指示灯、蜂鸣器等信号元件

### ⚠️ 注意事项

1. **类型名称**: 必须与 `COMPONENT_GRAPHICS` 中的键完全匹配
2. **未定义类型**: 如果类型未在 `COMPONENT_GRAPHICS` 中定义，会自动回退为直接连线
3. **节点位置**: 中间节点位置由draw.io自动布局，暂不支持手动指定

## 效果预览

### 生成的文件

```
scripts/examples/output/
├── DG_DEV1_..._CMP_SW1,SW2.drawio
│   └── 包含刀开关和闭合开关的接线图
└── DG_DEV3_..._CMP_LED1,SW2.drawio
    └── 包含LED指示灯的接线图
```

### 在draw.io中打开

打开生成的 `.drawio` 文件，可以看到：

- 🔵 端子显示为小圆点 + 文本标签
- ⚡ 开关显示为electrical库中的开关图形
- 💡 LED显示为灯泡图形
- ➡️ 连线自动连接各个节点

## 总结

互联类型功能通过在Excel数据中添加一个简单的列，实现了：

✅ **自动化**: 无需在draw.io中手动添加元件  
✅ **标准化**: 使用统一的图形库  
✅ **可追溯**: 连接类型记录在源数据中  
✅ **可扩展**: 轻松添加新的元件类型  
✅ **向后兼容**: 不影响现有数据和功能  

这大大提高了接线图绘制的效率和准确性！
